# SmartHire AI

An AI-powered recruitment platform built using FastAPI and PostgreSQL.

## Features

* Candidate Registration
* Candidate Login Authentication
* View Candidate Profile
* Update Candidate Profile
* Delete Candidate Profile
* PostgreSQL Database Integration
* Swagger API Documentation

## Tech Stack

* Python
* FastAPI
* PostgreSQL
* SQLAlchemy
* Pydantic
* Uvicorn
* Git & GitHub

## API Endpoints

### Candidate Management

* POST `/register`
* POST `/login`
* GET `/candidate/{candidate_id}`
* PUT `/candidate/{candidate_id}`
* DELETE `/candidate/{candidate_id}`
